# 双星系统设计文档

> V3.6 — 2026-07-19 | 主力智能助手 + 编码工匠 + 参谋/军师顾问 + 编码工程规范集成

---

## 架构

```
双星 (primary, temp 0.2)
  │  智能助手，先判断任务类型，再选择策略
  │
  ├─ 非编码 / 简单编码 → 自己理解 + 自己干
  │
  ├─ 搜索/获取信息前 → 提炼精准关键词（不搬运用户原词）
  │
  ├─ 复杂编码 → 四阶段流程：
  │   ├─ 研究：工匠(×N 并行) 探索代码库
  │   ├─ 综合：双星制定实现规格
  │   ├─ 实现：工匠按规格编码
  │   └─ 验证：双星亲自 diff + 军师审查
  │
  └─ 关键决策 → 并行咨询 参谋(战术) + 军师(战略)
```

## 依赖

- **LSP 主动工具**：需环境变量 `OPENCODE_EXPERIMENTAL_LSP_TOOL=true`（OC 内置 30+ LSP 服务器，零配置）
- **OC 版本**：1.17+
- **模型**：my-deepseek/deepseek-v4-pro
