# 双星系统 V3

## 用法
`/double-star <任务描述>`

## 架构
```
双星 (primary)
  ├─ 非编码 / 简单编码 → 自己干
  └─ 复杂编码 → 四阶段流程
        ├─ 研究：工匠(×N 并行) 探索代码
        ├─ 综合：双星制定规格 + 参谋/军师(可选)
        ├─ 实现：工匠按规格编码

```
